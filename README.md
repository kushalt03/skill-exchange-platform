# Skill Exchange Platform

A peer-to-peer learning platform where users can teach and learn skills from each other.

## Features

- User registration & profile management
- Skill listing (offer & learn)
- Skill-based search
- Connection system
- Chat system
- Session scheduling
- Rating system

## Tech Stack

- Backend: Java, Spring Boot
- Database: MySQL
- API Testing: Postman

## Core Modules

- User Management
- Skill Management
- Connection System
- Messaging System
- Session & Rating

## Future Enhancements

- Real-time chat (WebSocket)
- Video calling integration
- AI-based matching
- Notifications system

## Project Structure

- controller
- service
- repository
- entity

##  How to Run

1. Clone repo
2. Configure MySQL in `application.properties`
3. Run Spring Boot application
